Website Corporativo

Design e desenvolvimento de site responsivo para empresa de tecnologia

Sobre o Projeto

Este projeto envolveu o redesign completo do site corporativo de uma empresa de tecnologia, com foco em melhorar a experiência do usuário e otimizar a conversão de leads.

O design foi desenvolvido com uma abordagem mobile-first, garantindo uma experiência perfeita em todos os dispositivos. Implementamos animações sutis e microinterações para tornar a navegação mais envolvente.

O site foi construído com HTML5, CSS3 e JavaScript, utilizando técnicas modernas de desenvolvimento web para garantir performance e acessibilidade.

Duração: 2 meses
Cliente: TechCorp
Função: Design e Desenvolvimento

Aplicativo de Finanças

Interface de usuário para aplicativo de gestão financeira pessoal

Sobre o Projeto

Este projeto consistiu no design de interface para um aplicativo de gestão financeira pessoal, com foco em simplicidade e facilidade de uso.

O design foi desenvolvido no Figma, com protótipos interativos para testar a experiência do usuário. Trabalhamos com uma paleta de cores que transmite confiança e segurança, essenciais para um aplicativo financeiro.

Além do design visual, também desenvolvi a arquitetura de informação e fluxos de usuário, garantindo uma navegação intuitiva e eficiente.

Duração: 6 semanas
Cliente: FinApp
Função: UI/UX Design
